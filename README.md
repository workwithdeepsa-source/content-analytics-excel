# Content Performance Analysis using Excel

This project analyzes Instagram content performance using Microsoft Excel.

## Objectives
- Measure engagement rate across posts
- Compare performance by content format
- Analyze reach by posting day

## Tools Used
- Microsoft Excel
- Pivot Tables
- Formulas
- Charts

## Key Insights
- Reels generated the highest average reach.
- Carousel posts showed higher save behavior.
- Posting day influenced reach consistency.
